# airbnb-clone

Coding Airbnb clone using Python, Django and Tailwind

Features

    User Registration / Profile
    Social Authentication
    Superuser Admin Panel
    Host Admin Panel
    Guest Dashboard
    2-way Review System
    Bookin and Reservation System
    Direct Messaging
    Room CRUD
    Notification Email
    Ajax Searching

Technologies

    Python
    Django
    HTML5
    Tailwind CSS
    Gulp
    ES6
    Google Maps
    Twilio
    Mailgun
    Boto
